🔍 Duplicate Finder – Поиск дубликатов файлов на 7 языках
Мощный инструмент для обнаружения и управления дублирующимися файлами.
Использует комбинацию размера и криптографических хешей (MD5/SHA1) для точного сравнения.
Поддерживает большие файлы, исключения, симуляцию, удаление, перемещение и создание жёстких ссылок.
Реализован на семи языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Двухэтапное сравнение – сначала группировка по размеру, затем вычисление хеша для потенциальных дубликатов.

Быстрое хеширование – для больших файлов можно ограничиться чтением первых N байт (опция --partial).

Поддержка алгоритмов – MD5 (по умолчанию) или SHA1.

Гибкие действия:

--delete – удалить все дубликаты, оставив по одному экземпляру.

--move <dir> – переместить дубликаты в указанную папку.

--hardlink – заменить дубликаты жёсткими ссылками на первый файл (экономия места).

Режим симуляции (--dry-run) – показать, что будет сделано, без изменений.

Исключения (--exclude <pattern>) – исключить файлы/папки по glob-шаблону.

Рекурсивный обход – по умолчанию.

Логирование (--log <file>) – сохранить отчёт в файл.

Цветной вывод – наглядное отображение групп дубликатов.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <путь> [опции]
Основные опции
Опция	Описание
-p, --path <dir>	Папка для сканирования (по умолчанию текущая)
-r, --recursive	Рекурсивный обход (включен по умолчанию)
-a, --algo md5|sha1	Алгоритм хеширования (по умолчанию md5)
--partial <bytes>	Хешировать только первые N байт каждого файла (ускоряет для больших файлов)
-e, --exclude <pattern>	Исключить файлы/папки по glob-шаблону
--delete	Удалить дубликаты (оставить по одному)
--move <dir>	Переместить дубликаты в указанную папку
--hardlink	Заменить дубликаты жёсткими ссылками на оригинал
-n, --dry-run	Только показать, что будет сделано
-l, --log <file>	Сохранить отчёт в файл
-v, --verbose	Подробный вывод (показывать хеширование)
-h, --help	Справка
Примеры
bash
# Найти все дубликаты в ~/Documents
python find_duplicates.py ~/Documents

# Использовать SHA1 и показать подробности
python find_duplicates.py -a sha1 -v ~/Downloads

# Удалить дубликаты в папке проекта (без подтверждения)
python find_duplicates.py ./project --delete -y

# Переместить дубликаты в папку /tmp/dups и сохранить лог
python find_duplicates.py /data --move /tmp/dups -l report.log

# Только показать, что будет удалено (симуляция)
python find_duplicates.py ~/Pictures --delete -n

🧠 Логика работы
Обход файлов – рекурсивный проход по указанной папке (с учётом исключений).

Группировка по размеру – файлы с уникальным размером не могут быть дубликатами.

Хеширование – для групп с одинаковым размером вычисляется хеш (MD5 или SHA1) всего файла или первых N байт.

Группировка по хешу – файлы с одинаковым хешем считаются дубликатами.

Вывод результатов – список групп дубликатов с указанием размера и путей.

Действия – удаление, перемещение или создание жёстких ссылок (по желанию).

Все действия можно предварительно просмотреть в режиме --dry-run.

✨ Дополнительные фичи
Работа с большими файлами – хеширование по частям без загрузки всего файла в память.

Пропуск системных файлов – автоматически игнорируются .DS_Store, Thumbs.db и т.п.

Цветной вывод – группы дубликатов выделяются, размер показывается в удобном формате.
